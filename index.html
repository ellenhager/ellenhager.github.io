Bachelor Project

Course Syllabus

The project created for the bachelor thesis was a game for a dome theatre. Programmed in C++, using SGCT (Simple Graphics Cluster Toolkit), a library created at Visualiseringscenter C in Norrköping, and OSG (OpenSceneGraph). The game is set in space and is a multiplayer cooperative strategy game.

The source code for the game can be found here

TDDD23 Design and Programming
of Computer Games

Course Syllabus

The aim of this course is to create a game using an existing game engine. The game I created was made using Unity2D and is a simple 2D game. The aim of the game is to jump as high up as possible and collect as many points as possible.

TNCG13 SFX - Tricks of the Trade

Course Syllabus

The project for this course was to implement a plugin for Autodesk Maya. My group created a soft body simulation using pressure models. The plugin was written in C++. Source code available here

TNM091 Media Production
for Immersive Environments

Course Syllabus

This project was created for a dome theatre, in an attempt to create a fun and simple game that could be played my the entire audience before a film show. The game is controlled by the volume of the audience, and the aim is to solve a simple puzzle, getting the audience to work together. The game was made in OpenGL C++, and also utilising SGCT.

The source code for the game can be found here, video hopefully available soon!
